Как использовать кадастровые данные

Для работы с кадастровыми участками:

Важно! Кадастровые данные загружаются только для текущего региона карты
  1. Перейдите к нужному региону на карте
  2. Включите "Показать кадастровые участки"
  3. Дождитесь загрузки данных (может занять 10-30 секунд)
  4. Найдите ваш участок и запомните кадастровый номер
  5. Используйте его при добавлении нового поля
Подсказка:

Если данные не загружаются:

  • Попробуйте изменить масштаб карты
  • Подождите 1-2 минуты и повторите попытку
  • Проверьте правильность кадастрового номера
