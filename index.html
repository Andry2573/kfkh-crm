Инструкция по использованию:

  1. Нажмите на карту, чтобы установить точки границ поля
  2. После завершения контура нажмите "Сохранить поле"
  3. Заполните информацию о поле и посеве
  4. Данные сохранятся автоматически
  5. Для синхронизации с Яндекс.Диском нажмите кнопку "Синхронизировать"

  3. Использовать ручной ввод данных
